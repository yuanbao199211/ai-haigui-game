# AI 海龟汤：深渊真相 技术设计文档

## 1. 技术栈
- 前端框架：React 18
- 样式方案：Tailwind CSS
- 图标库：Lucide React
- 核心模型：Gemini 2.5 Flash API

## 2. 状态架构
### 核心状态管理
- `screen`: 页面路由状态（Lobby / Game / Result）。
- `mentalFocus`: 资源点状态，控制提问权限。
- `clues`: 提取出的关键情报数组。
- `messages`: 聊天历史对象。

## 3. 关键算法与逻辑
### 3.1 文本渲染算法
- 打字机组件：使用 `text.slice(0, i + 1)` 模式处理中文字符，防止索引偏移导致的文字丢失。

### 3.2 线索提取逻辑
- 自动提取：监听 AI 响应，匹配关键字（如“是”、“关键”），将对应的用户问题推送到情报库。

### 3.3 API 通信
- 指数退避重试：实现 5 次请求重试机制（延迟：1s, 2s, 4s, 8s, 16s），确保在 API 限制下的高可用性。

## 4. 视觉表现层
- 背景光效：使用固定定位的 CSS 模糊滤镜层，减少性能消耗。
- Eureka Flash：触发时在整个页面顶层显示一个短暂的 `opacity-10` 红色脉冲 div。

## 5. 环境配置
- API Key：通过运行时环境变量注入。
- 模型参数：设置 `temperature` 为 0.7 以平衡创意与逻辑稳定性。
